Dưới đây là code C++ thuật toán Prim chuẩn và dễ hiểu, dùng priority_queue (min-heap) để lấy cạnh nhỏ nhất nhanh.

---

✅ C++ – Prim Algorithm (Adjacency List + Min Heap)

#include <bits/stdc++.h>
using namespace std;

typedef pair<int, int> pii; 
// first = weight, second = node

const int INF = 1e9;

int main() {
    int n, m;
    cin >> n >> m;

    vector<vector<pii>> adj(n + 1);

    // Nhập danh sách cạnh
    for (int i = 0; i < m; i++) {
        int u, v, w;
        cin >> u >> v >> w;
        adj[u].push_back({w, v});
        adj[v].push_back({w, u});
    }

    vector<bool> used(n + 1, false);
    priority_queue<pii, vector<pii>, greater<pii>> pq;

    // Bắt đầu từ đỉnh 1
    pq.push({0, 1});

    int mst_weight = 0;

    while (!pq.empty()) {
        auto [w, u] = pq.top();
        pq.pop();

        if (used[u]) continue;
        used[u] = true;
        mst_weight += w;

        // Duyệt cạnh kề u
        for (auto &edge : adj[u]) {
            int weight = edge.first;
            int v = edge.second;
            if (!used[v]) {
                pq.push({weight, v});
            }
        }
    }

    cout << "MST Weight = " << mst_weight << "\n";

    return 0;
}

---

📌 Giải thích nhanh

Dùng priority_queue để luôn lấy cạnh nhỏ nhất.

Mảng used[] để đánh dấu node đã vào MST.

Mỗi lần chọn đỉnh mới → cộng trọng số → đẩy các cạnh xuất phát từ đỉnh đó vào heap.
